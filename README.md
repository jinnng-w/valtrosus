# Valtros Static Website

A responsive, single-page company website for Valtros Inc., focused on solar mounting structures and Texas manufacturing.

## Preview

Open `index.html` directly in a browser. No build step or third-party framework is required.

## Project files

- `index.html` — page content and structure
- `styles.css` — visual system, responsive layouts and animation
- `script.js` — mobile navigation and scroll reveals
- `images/` — local logo, solar, factory and product photography

## Before publishing

Confirm the company name, capacity figures and both Sealy addresses. Then upload this folder to a GitHub repository and enable GitHub Pages.
